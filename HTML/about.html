Website Rational

Cost Of Living Comparator is a modern website that allows users to compare the different costs of living within capital cities all around the world. The website gives the user a wide variety of capitals to choose from. The user is expected to select two different capitals (one being the base town, and the second one, the compare town).

The website is then responsible for retrieving all relevant information about the capitals that the user has selected. This includes, but is not limited to, the name of the country, the corresponding capital city, the population, the Longitude and latitude of the capital so it can be found on the map, and a flag of the country. The website also uses APIs to retrieve information on the cost of living base on either education, food, transport, housing or utilities. It then uses modern libraries such as Chart.js to graphically represent the information received.
